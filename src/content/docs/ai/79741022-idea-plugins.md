## 🚀 安装与配置

### 插件安装步骤

1. **打开插件市场**: `File → Settings → Plugins`
2. **搜索插件名称**: 使用插件名称精确搜索
3. **安装插件**: 点击"Install"按钮
4. **重启IDE**: 部分插件需要重启生效

### 插件管理最佳实践

- **定期更新**: 保持插件版本与IDE同步
- **性能监控**: 注意插件对启动速度的影响
- **冲突避免**: 安装前查看用户评价和已知问题
